Te damos la bienvenida a este proyecto

git init-----> inicia el repositorio dentro de la carpeta del proyecto

git add ----> agrega las cosas nuevas a la caja (puede recibir un parametro)... add . guarda todo los archivos modificados dentro del repositorio en la caja

git status ----> nos dice el estado acutal del repositorio, si hay cajas abieras q esperan ser cerradas para generar una nueva version

git commit -m "version 1" -----> cerraria la caja y generia una version del proyecto pudiendo ponerle un mensaje recibiendo un parametro para el mensaje

git log -----> muestra las versiones disponibles dentro del proyecto

git log --oneline ------> muestra lo mismo que git log de manera mas acotada

git remote add "nombre del repositorio por lo general origin" y la url----> agregar un repositorio remoto, hay q integrar el link del repositorio remoto luego del origin

git push origin main------> permite llevar de las versiones de manera local al remoto, lleva dos parametros
